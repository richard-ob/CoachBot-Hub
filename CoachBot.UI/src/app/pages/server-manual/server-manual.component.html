Server Manual

Introduction

This manual is primary aimed at those wishing to host a dedicated IOSoccer server. If you have any queries, or issues, please feel free to ask in #helpdesk on our Discord Server.

Common Server console commands and variables

Note: If you execute these commands from the client console you need to set rcon_password x in cfg\server.cfg and write rcon before the actual command.

  • exec x.cfg: Executes a config file in the cfg folder named x.
  • sv_restart x: Resets the match and goes into warm up mode with an x minutes countdown. Use -1 to stay in warm up mode forever.
  • mp_maxplayers x: Sets x number of players per side.
  • benchall 0: Move all players from field to bench.
  • mp_botkeepers x: 1 to enable AI keepers, 0 for human keepers.
An exhaustive list is available at the bottom of this page
Windows dedicated server installation example
  1. Download SteamCMD and extract steamcmd.exe to a folder
  2. Open the folder in Explorer, hold shift, right-click in an empty area and click Open command/PowerShell window here. Enter the following commands in the terminal:
  3. .\steamcmd +login anonymous +force_install_dir iosoccer +app_update 673990 validate +quit
  4. cd iosoccer
  5. .\srcds.exe -console -game iosoccer -condebug -port 27015 +maxplayers 32 +fps_max 500 -tickrate 100 +map 8v8_vienna
    You may need to use +ip xxx.xxx.xxx.xxx if the server has multiple network adapters, or is a more complex setup, otherwise this is not necessary.

The server should now be running. You can enter server commands in the console.

Linux dedicated server installation example
  1. See here for instructions on how to install SteamCMD on Linux.
  2. steamcmd +login anonymous +force_install_dir iosoccer +app_update 673990 validate +quit
  3. cd iosoccer
  4. ./srcds_run -console -game iosoccer -condebug -port 27015 +maxplayers 32 +fps_max 500 -tickrate 100 +map 8v8_vienna
    You may need to use +ip xxx.xxx.xxx.xxx if the server has multiple network adapters, or is a more complex setup, otherwise this is not necessary.

The server should now be running. You can enter server commands in the console.

Enabling custom assets

If you want to use custom assets on your dedicated server you need a fastdl setup on a webserver, because sending the files to players directly through the game server is extremely slow.

To enable custom assets open iosoccer\cfg\autoexec.cfg and change sv_custom_assets 0 to sv_custom_assets 1.

Server Commands
Command Example Information
rcon_password rcon_password serverpassword Used to gain access to the server's administration commands.
rcon rcon command Used to invoke rcon for usage of admin commands in the server.
quit rcon quit Restarts the server if a problem exists in it.
mp_teamkits rcon mp_teamkits 1 2 Sets specific team kits. The first number sets the kit for the home team and the second one sets it for the away team. You can also search for specific team kits by doing mp_teamkits teamname.
sv_restart rcon sv_restart 1 Sets a timer to start the match. It is measured in minutes, so sv_restart 1 makes the match start in 1 minute, 0.5 in half a minute, 0 starts the match imediately and -1 sets the server to training mode.
sv_singlekeeper rcon sv_singlekeeper 1 Activates/Disables single keeper mode in the server. 1 activates it and 0 disables it.
bench rcon bench name Benches a player through their in-game name.
status rcon status Displays information about the players currently in the server, including IP, SteamID and game ID.
benchid rcon benchid 7 Benches a player determined on their GameID, obtainable through the "status" command.
benchall rcon benchall 0 Benches every player in the server. To do it, you must type rcon benchall 0.
say rcon say Hello Displays a message in the universal chat of the server as "Stadium announcer".
sv_alltalk rcon sv_alltalk 1 Enables/Disables global voice chat. 1 enables it and 0 disables it.
sv_custom_assets rcon sv_custom_assets 1 Enables/Disables the usage of custom assets in the server. 1 enables it and 0 disables it
sv_gravity rcon sv_gravity 900 Modifies the ammount of gravity in the server. Default value is 900.
mp_shield_block_opponent_half rcon mp_shield_block_opponent_half 0 Used to enable or disable the midfield shield. 1 enables it 0 disables it.
exec rcon exec 8v8 Executes a .cfg file. To execute a certain type of match use rcon exec yxy, to execute official matches use rcon exec league or rcon exec cup, accordingly.
sv_matchminute rcon sv_matchminute 45 Sets the match timer to a specific minute in the match.
sv_matchgoalshome rcon sv_matchgoalshome 3 Sets the number of goals the home team has on the scoreboard.
sv_matchgoalsaway rcon sv_matchgoalsaway 2 Sets the number of goals the away team has on the scoreboard.
sv_endmatch rcon sv_endmatch Ends an ongoing match.
mp_maxplayers rcon mp_maxplayers 8 Defines the number of players allowed on each side.
sv_starttimeout rcon sv_starttimeout Starts a forced timeout.
sv_endtimeout rcon sv_endtimeout Ends a forced timeout.
mp_penalties rcon mp_penalties 1 Enables/Disables penalty shootout in a match. 1 enables it and 0 disables it.
mp_extratime rcon mp_extratime 1 Enables/Disables extra time in a match. 1 enables it and 1 disables it.
sv_wakeupcall rcon sv_wakeupcall Sends a message on the server alerting players to move.
sv_removeplayercardban id rcon sv_removeplayercardban 12 Removes a player's red card ban through their GameID
sv_replay_instant_enabled rcon sv_replay_instant_enabled 1 Enables/disables replays after a goal is scored.
sv_playerrotation rcon sv_playerrotation 0 Enables/Disables player rotation through positions. 1 enables it and 0 disables it.
sv_propcreation rcon sv_propcreation 1 Enables/Disables prop creation in the server. 1 enables it and 0 disables it.
sv_physicsprops rcon sv_physicsprops 1 Enables/Disables physics for props. 1 enables it and 0 disables it.
sv_resumematch rcon sv_resumematch Resumes the ongoing match.
sv_cheats rcon sv_cheats 1 Allows/Disallows cheat commands. 1 enables it and 0 disables it.
sv_bots_fill_field rcon sv_bots_fill_field 1 Fills every available position on the field with bots.
mp_humankeepers rcon mp_humankeepers 1 Enables/Disables human GKs. 1 enables it and 0 disables it.
mp_botkeepers rcon mp_botkeepers 0 Enables/Disables bot GKs. 1 enables it and 0 disables it.
mp_timelimit_match mp_timelimit_match 20 Defines the match time in minutes. For 4v4 the default is 10 and for 8v8 the default is 20. Maximum is 90.
mp_timelimit_part rcon mp_timelimit_part 20 There are various instances to replace part, and they are: match, warmup, cooldown, halftime, extratime_halftime, extratime_intermission and penalties_intermission
mp_goldengoal rcon mp_goldengoal 1 Enables/Disables golden goal mode. 1 enables it and 0 disables it.
mp_offside rcon mp_offside 0 Enables/Disables offsides. 1 enables it and 0 disables it.
mp_serverinfo rcon mp_serverinfo NTA's home Sets the server info. Server info is visible upon pressing tab and looking at the top left corner. Default in IOS servers is ""IOSoccer Server - Enjoy your stay""
mp_timeoutcount rcon mp_timeoutcount 1 Defines how many times a team can call for a timeout during a match. The default is 3 for friendlies and mix matches.
mp_timeout_duration rcon mp_timeout_duration 3 Defines how long a timeout can take, during all it's calls. The default value is 5.
mp_botkeeperspeed rcon mp_botkeeperspeed 90 Defines botkeepers' speed, from a range of 0 to 100. Default is 100.
mp_botkeeperdelay rcon mp_botkeeper_delay 40 Defines botkeepers' response time in ms. Default is 0.
mp_player_model_scale id rcon mp_player_model_scale 23 200 Defines a player's size from a range of 0-255 (being the default 155) through their GameID
mp_chat_match_captain_only rcon mp_chat_match_captain_only 1 Enables/Disables captain exclusive chat. 1 enables it and 0 disables it.
mp_chat_intermissions_captain_only rcon mp_chat_intermissions_captain_only 1 Enables/Disables captain exclusive chat during intermissions. 1 enables it and 0 disables it.
mp_maxprops rcon mp_maxprops 20 Defines how many props can be spawned by a single player.
